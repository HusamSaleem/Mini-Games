Summary: This program has mini-games such as tic tac toe, rock paper scissors, and guessing a number from a randomly generated list. Basically a mini-game program.
		 -> The Game Class is the abstract class that was used as a base for the Tic Tac Toe, rock paper scissors and the guessing number sub-class. The ScoreSystem was a interface the was made to make sure that the player gets
		 points. Then the GameManager class was basically the class that handled all the input of the player, creating an account, logging in, etc...
		 
**HOW TO RUN**

1.Run the GameManager.java class
2. Create your account by pressing "1" in the console
3. Enjoy playing simple games!
